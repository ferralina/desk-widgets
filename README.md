## 💡 我们做了什么

把**待办、日历、天气、便签、阶段目标、桌面文件夹**六个常用功能，
做成**贴在桌面上**的小窗口。

就像桌面上的便签纸和效率工具，**随取随用，不用就收起来**。

### 可以解决的具体场景

| 场景 | 以前 | 现在 |
|------|------|------|
| 看今天有啥待办 | 打开 Todoist/滴答清单 | 瞥一眼桌面角落 |
| 查天气 | 打开浏览器搜 | 桌面右上角直接看 |
| 写便签 | 贴墙上/开 Notepad | 随手开个便签窗口 |
| 记目标 | 在脑子里/Excel | 桌面底部进度条 |
| 找文件 | 在文件夹里翻 | 从桌面启动器直接搜 |
| 多任务切换 | 任务栏里找 | 组件都在浮窗上 |

## 🎨 效果预览

所有组件统一风格：**磨砂玻璃 + 无边框 + 自由拖拽**

- 拖到屏幕边缘自动贴边，半透明隐藏，鼠标悬停恢复
- 透明度可以调，想清晰就清晰，想隐身就隐身
- 位置、状态全部**自动记住**，重启也不丢
- **多屏支持**，每个屏幕都可以放组件

## 🛠 用 Cursor / OpenClaw 二次开发

**面向有编程基础的同学：**

```bash
# 克隆下来
git clone https://github.com/ferralina/desk-widgets.git
cd desk-widgets
npm install
npm start
```

修改 `widgets/` 下的任意 `index.html`，保存后窗口自动热更新。
用 **Cursor** 或 **OpenClaw** 的 Agent 模式，直接用自然语言让它帮你改：

> "帮我把天气改成显示上海"
> "待办组件加上截止时间提醒"
> "便签改成支持多个便签"

**架构足够简单，改起来很快。**

## ⚙️ 技术细节

| 项目 | 技术 |
|------|------|
| 框架 | Electron ^41 |
| 拖拽 | SortableJS ^1.15 |
| 天气 | wttr.in（无需 API Key） |
| 窗口 | 无边框 + 透明 + 磨砂玻璃 |
| 持久化 | 本地 JSON 文件 |
| 语言 | HTML / CSS / JS（无框架） |
